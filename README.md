  # Computer Vision 電腦視覺

  長庚大學電腦視覺課程的作業與課堂練習紀錄。

  ## 作業內容（HW）

  | 作業 | 主題 | 使用技術 |
  |------|------|----------|
  | HW2 | 影像資料視覺化與分析 | Python、Matplotlib、NumPy |
  | HW3 | 臉部關鍵點偵測（CNN） | PyTorch、CNN、MSE Loss |
  | HW4 | 物件偵測（RetinaNet） | PyTorch、RetinaNet、Oxford-IIIT Pet Dataset |
  | HW5 | 3D 點雲配準（RANSAC + ICP） | Open3D、RANSAC、ICP |

  ## 課堂練習（Practice）

  | 練習 | 主題 | 使用技術 |
  |------|------|----------|
  | Practice 1 | 影像處理多子圖視覺化 | Matplotlib |
  | Practice 2 | 臉部關鍵點偵測（FCN） | PyTorch、FCN、SGD |
  | Practice 3 | 臉部關鍵點偵測（CNN） | PyTorch、CNN、Adam |
  | Practice 4 | 手寫數字識別 + 人臉偵測 | OpenCV、HOG、SVM、Haar Cascade |
  | Practice 5 | Open3D 函式庫介紹 | Open3D |
  | Practice 6 | 3D 點雲表面重構（Ball Pivoting） | Open3D、Ball Pivoting Algorithm |
  | Practice 7 | 多視角 Voxel Carving | Open3D、Voxel Grid |
